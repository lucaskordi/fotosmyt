Cole este CSS no campo "CSS Personalizado" da Yampi:

===========================================

label:after {
  display: none !important;
}

span:after {
  display: none !important;
}

*[class*="optional"],
*[id*="optional"] {
  display: none !important;
}

label,
span {
  position: relative;
}

label::after,
span::after {
  content: "" !important;
  display: none !important;
}

===========================================
